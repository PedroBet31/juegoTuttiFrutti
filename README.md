# **Tutti Frutti Web**

Tutti Frutti Web is a web application developed as an academic project based on the classic word game **Tutti Frutti**. The system allows users to register, create or join game rooms, and compete by completing different categories with words that start with a randomly generated letter.

The application includes both **single-player and multiplayer modes**, with automatic validation of answers and score calculation for each round.

## **Main Features**

User registration and login  
 Main lobby to access the game modes  
 Single player mode  
 Multiplayer rooms (create or join matches)  
 Game room where players wait before starting  
 Random letter generation for each round  
 Category-based word answers  
 Answer validation system  
 Partial results after each round  
 Final ranking at the end of the match

## **Technologies Used**

Java  
 Spring Boot  
 HTML  
 CSS  
 JavaScript  
 MVC architecture
